## [0.1.0] - 2026-01-07

### Added
- Initial release of marty-rs (marty-credentials Rust bindings)
- Status list implementation for credential revocation
- Python bindings via PyO3 (abi3-py311)
- WebAssembly support via wasm-bindgen
- Integration with marty-crypto for cryptographic operations
- SSI (Self-Sovereign Identity) credential support
